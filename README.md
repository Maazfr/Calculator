📱 Calculator
A sleek, responsive calculator web app inspired by the iPhone's native calculator. Built using HTML, CSS, and JavaScript, this project offers basic arithmetic functionalities within a clean and intuitive interface.


🚀 Features
Basic Operations: Addition, subtraction, multiplication, and division.

Responsive Design: Optimized for desktops, tablets, and mobile devices.

iPhone-Inspired UI: Mimics the look and feel of the iOS calculator.

Keyboard Support: Allows input via keyboard for enhanced usability.

Clear Functionality: Easily reset calculations with a clear button.

🛠️ Technologies Used
HTML5: Structuring the calculator's layout.

CSS3: Styling and responsive design.

JavaScript (ES6): Handling calculator logic and interactivity.
